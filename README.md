# Agri-Yield

Agri-Yield is a fintech + agritech platform designed to connect Ethiopian farmers, investors, merchants, and off-takers through secure digital agriculture financing.

## Core Features

- Farmer registration & KYC
- Farm management
- Investment marketplace
- Escrow system
- QR voucher payments
- Satellite NDVI monitoring
- AI-powered advisory
- Fraud detection

## Tech Stack

### Backend
- Java 21
- Spring Boot
- Spring Cloud
- PostgreSQL
- MongoDB
- Redis
- RabbitMQ

### Frontend
- Next.js
- TypeScript
- TailwindCSS

### Mobile
- Flutter

## Architecture
Microservices architecture using:
- API Gateway
- Eureka Service Discovery
- RabbitMQ Event Bus
- Docker Infrastructure
